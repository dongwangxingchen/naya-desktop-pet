# 角色方向稿说明

这三张图用于锁定重置版的整体风格、服饰细节与慵懒姿态方向，不直接当作逐帧动画素材。

- `naya-lazy-pose-01.png`：候选默认坐姿，适合靠近屏幕边缘停留。
- `naya-lazy-pose-02.png`：更放松的半靠姿势，可作为低频待机动作参考。
- `naya-lazy-pose-03.png`：夜间困倦动作参考；角色仍保持可唤醒、可交互。

正式进入程序前还需要：

1. 确认最终默认姿势与角色尺寸。
2. 制作透明背景的统一母版。
3. 按固定锚点重新制作身体、头部、眼睛、眼皮、瞳孔、头发、辫子和饰品图层。
4. 通过眨眼、视线、呼吸和头部微动测试后，才制作更多大动作。
